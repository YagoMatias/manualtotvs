MÓDULO: ACESSANDO O AMBIENTE

1. Abertura do Sistema

  1. Abra o sistema oficial da CROSBY:
    • Acesse pela plataforma disponibilizada pela empresa
    • Utilize o atalho na área de trabalho quando disponível

2. Inicialização do Virtual AGE

  1. Na tela inicial:
    • Localize o botão "VIRTUAL AGE"
    • Clique para iniciar o acesso ao sistema

3. Autenticação

  1. Insira suas credenciais:
    • Digite seu usuário no campo correspondente
    • Informe sua senha no campo designado
    • Clique no botão "Entrar"

4. Seleção da Empresa

  1. Após login:
    • Selecione a empresa/franquia que você representa
    • Escolha a unidade vinculada ao seu usuário

5. Acesso ao Módulo de Vendas

  1. No menu principal:
    • Localize e clique em "Virtual Store SALES"
    • Aguarde o carregamento do módulo

6. Configuração do Terminal

  1. Quando solicitado:
    • Informe o número do terminal como 1
    • Pressione Enter ou clique em Confirmar

Observações Importantes:

  • O terminal 1 é padrão para operações
  • Em caso de erro no login, verifique o Caps Lock e o idioma do teclado
  • Para problemas de acesso, contate o suporte técnico
  • Não compartilhe sua senha com outros colaboradores

MÓDULO: ENTENDENDO O AMBIENTE

1. Componentes (ao centro da tela)

No centro do sistema, você verá vários quadrados, chamados de componentes.

  • Cada componente possui uma função específica
  • São identificados por um código único
  • Exemplo: O componente TRAFL016 é um dos códigos disponíveis

2. Menu Auxiliar (lateral esquerda)

Localizado à esquerda da tela, o Menu Auxiliar permite:

  • Filtrar componentes por funcionalidades
  • Agrupar por categorias de uso
  • Exemplos de filtros disponíveis: CAIXA, VENDAS, PESSOA...

3. Acessando os Componentes

Existem duas formas de acessar os componentes:

  1. Clique direto no ícone do componente desejado
  2. Digite o código na área:
    • Acesse Ferramentas > Componentes
    • Localizado na parte superior da tela

MÓDULO: ABRIR CAIXA

1. Acesso ao Menu de Vendas

  1. No menu auxiliar localizado na lateral esquerda:
    • Clique na categoria "VENDAS"
    • Isso exibirá todos os componentes relacionados

2. Abertura do PDV

  1. No menu de componentes:
    • Localize e clique em "Venda Varejo - PDVFM001"
    • Este é o módulo principal de frente de loja
  2. Após abertura do PDV:
    • Acesse o Menu Caixa

3. Abertura do Caixa

  1. No Menu Caixa:
    • Clique em "Abrir Caixa"
    • Confirme a ação clicando em "Confirmar"
  2. O sistema:
    • Iniciará o caixa do dia
    • Habilitará todas as funções de vendas
    • Registrará o horário de abertura

Observações Importantes:

  • Verifique se o caixa anterior foi devidamente fechado
  • A abertura de caixa deve ser a primeira operação do dia
  • O sistema registra automaticamente o usuário que realizou a abertura
  • Não é possível registrar vendas sem antes abrir o caixa

MÓDULO: VENDAS

1. Acesso ao PDV

  1. Acesse o componente PDVFM001 – Venda Varejo:
    • Este é o módulo principal para registro de vendas
    • Interface otimizada para operação rápida

2. Identificação do Vendedor

  1. No campo VENDEDOR:
    • Selecione seu código na lista
    • OU digite manualmente seu código

3. Identificação do Cliente

  1. No campo CLIENTE:
    • Busque pelo nome ou CPF
    • Para novos clientes, clique no "C"
    • Cliente "Balcão" é selecionado automaticamente

4. Inclusão de Produtos

  1. Para adicionar produtos:
    • Use a leitor de pistola para bipar o código
    • OU digite manualmente o código
    • Confirme cada item com TAB

5. Aplicação de Descontos

  1. Caso necessário aplicar desconto:
    • Clique no botão "Desconto"
    • Informe valor em R$ ou %
    • Confirme
    • Descontos podem exigir autorização do supervisor

6. Finalização da Venda

  1. Selecione a FORMA DE PAGAMENTO:
    • Dinheiro (calcula troco automaticamente)
    • Cartão (débito/crédito - selecione bandeira)
    • Pix
    • MIX
  2. Confirme a venda para emitir o cupom fiscal

Observações Importantes:

  • Verifique sempre o preço exibido no visor
  • Para produtos sem código, use a busca por descrição

MÓDULO: CADASTRAR CLIENTE

1. Preparação Inicial

  1. No ambiente do PDV:
    • Selecione o vendedor responsável
    • Esta seleção é obrigatória antes do cadastro
  2. Remova o cliente temporário:
    • Apague o cliente padrão "Balcão"
    • Limpe o campo de cliente balcão da tela principal

2. Início do Cadastro

  1. Clique no botão "C":
    • Isso abrirá a tela completa de cadastro
    • Novo formulário será exibido

3. Preenchimento dos Dados

  1. Informe os dados obrigatórios:
    • Nome completo
    • Telefone principal
    • Data de nascimento
    • Estado civil
    • Sexo (F/M/O)
    • E-mail válido
    • Endereço completo
    • CPF

4. Finalização do Cadastro

  1. Para salvar:
    • Clique no ícone 💾 (disquete)
    • O sistema validará os dados
  2. Para vincular à venda:
    • Clique no botão ✅ (confirmar)
    • O cliente será enviado ao PDVFM001

Observações Importantes:

  • Campos marcados com * são obrigatórios
  • Clientes cadastrados ficam disponíveis para futuras compras

MÓDULO: ALTERAR CLIENTE

1. Preparação Inicial

  1. No ambiente do PDV:
    • Selecione o vendedor responsável
    • Verifique se está no terminal correto

2. Acesso ao Cliente

  1. Clique no botão "Cliente...":
    • Localizado na área de informações do cliente
    • Abre a janela de pesquisa

3. Pesquisa do Cliente

  1. Realize a pesquisa:
    • Digite nome ou CPF
    • Clique no ícone Lupa
    • OU pressione Enter

4. Edição dos Dados

  1. Para editar:
    • Clique no botão "C" (Cadastro)
    • O sistema abrirá o formulário completo
  2. Altere os campos necessários:
    • Dados pessoais
    • Contatos
    • Endereço
    • Informações complementares

5. Salvamento e Confirmação

  1. Para salvar alterações:
    • Clique no ícone 💾 (disquete)
    • O sistema validará as modificações
  2. Para vincular à venda:
    • Clique no botão ✅ (confirmar)
    • O cliente será enviado ao PDVFM001
    • Os dados atualizados serão utilizados

Observações Importantes:

  • Alterações em CPF exigem autorização gerencial
  • O sistema registra data/hora e usuário da alteração

MÓDULO: APLICAR DESCONTOS

1. Acesso ao PDV

  1. Acesse o componente PDVFM001

2. Informações Básicas

  1. Preencha os dados obrigatórios:
    • Vendedor responsável
    • Cliente (se aplicável)
    • Produtos da venda

3. Acesso à Tela de Descontos

  1. Clique na opção "Desconto"

4. Configuração do Desconto

  1. Informe os parâmetros:
    • Valor em R$ (absoluto)
    • OU Percentual %
  2. Selecione o tipo:
    • Botão [ R$ ] para valor fixo
    • Botão [ % ] para percentual

5. Confirmação

  1. Clique em "Confirmar"
  2. O sistema:
    • Aplicará automaticamente o desconto
    • Rateará proporcionalmente entre os produtos

Observações Importantes:

  • Descontos acima de 10% exigem autorização
  • O sistema registra quem aplicou o desconto
  • Verifique o valor total após desconto

MÓDULO: FORMAS DE PAGAMENTO

1. Dinheiro

  1. Clique no botão Dinheiro (lateral direita)
  2. Informe:
    • Valor recebido do cliente
  3. O sistema:
    • Calcula troco automaticamente
  4. Clique em Confirmar

2. Débito

  1. Clique no botão Débito
  2. Selecione:
    • Operadora do cartão
    • Bandeira (Visa/MasterCard/etc.)
  3. Informe:
    • Número da autorização
    • NSU (comprovante)
  4. Clique em Confirmar

3. Crédito

  1. Clique no botão Crédito
  2. Selecione:
    • Operadora e bandeira
    • Número de parcelas
  3. Informe:
    • Autorização
    • NSU
  4. Clique em Confirmar

4. Mix de Pagamentos

  1. Clique no botão Mix
  2. Para cada forma:
    • Informe valor parcial
    • Selecione tipo (Dinheiro/Débito/etc.)
  3. Clique em Confirmar

Observações Importantes:

  • Verifique sempre os valores informados
  • Para cartões, confira os dados da maquininha
  • Mix permite combinar até 3 formas diferentes

MÓDULO: PIX

1. Acesso ao Mix

  1. Clique no botão MIX (lateral direita)

2. Configuração do PIX

  1. Informe:
    • Valor do PIX
  2. Selecione:
    • Opção "OUTROS DOCUMENTOS"

3. Dados Adicionais

  1. Preencha:
    • Documento: Código da transação
    • Data: Data atual

4. Confirmação

  1. Clique em Confirmar

Observações Importantes:

  • O comprovante do PIX deve ser guardado
  • Verifique o recebimento no app da empresa

MÓDULO: SAÍDA EM CONDICIONAL

1. Acesso ao PDV

  1. Acesse o componente PDVFM001 – Venda Varejo

2. Identificação dos Participantes

  1. Informe:
    • Vendedor responsável
    • Cliente que receberá os produtos

3. Ativação do Modo Condicional

  1. Clique na opção "Cond." (Condicional)

4. Inclusão de Produtos

  1. Adicione os produtos:
    • Via leitor de código de barras
    • OU busca manual

5. Finalização

  1. Selecione:
    • Forma de pagamento (ex: Dinheiro)
  2. O sistema:
    • Gerará comprovante de condicional
    • Registrará a saída no estoque

Observações Importantes:

  • Produtos condicionais têm prazo máximo de retorno
  • Verifique dados do cliente antes de confirmar
  • O sistema bloqueia produtos com restrição para condicional

MÓDULO: RETORNO DE CONDICIONAL

1. Acesso Inicial

  1. Acesse o componente PDVFM001 – Venda Varejo

2. Identificação

  1. Informe:
    • Vendedor responsável
    • Cliente condicional

3. Validação

  1. Na janela de confirmação:
    • Selecione "Sim"

4. Acesso ao Acerto

  1. Acesse o componente PDVFM017 – Acerto de Condicional

5. Processamento dos Produtos

  1. Para cada produto:
    • Bipe o código OU digite manualmente
    • Selecione ação:
      • Devolver ao estoque
      • OU "Faturar tudo"

6. Finalização

  1. Clique em Confirmar
  2. Selecione forma de pagamento (se necessário)

Observações Importantes:

  • Verifique estado dos produtos devolvidos
  • Produtos faturados geram NF-e e NFC-e automática

MÓDULO: TROCA PARA CLIENTE

1. Preparação

  1. Acesse o componente PDVFM001 – Venda Varejo
  2. Informe:
    • VENDEDOR responsável
    • CLIENTE que realizará a troca

2. Ativação do Modo Troca

  1. Clique no botão "TROCA"

3. Registro do Produto de Entrada

  1. Informe o PRODUTO DE ENTRADA:
    • Via leitor de código de barras
    • OU busca manual

4. Desativação do Modo Troca

  1. Clique novamente em "TROCA"

5. Seleção de Documento Fiscal

  1. Selecione:
    • NF-e para troca com nota fiscal
    • NFC-e para cupom fiscal

6. Finalização da Troca

  1. Selecione DINHEIRO para finalizar a transação

Observações Importantes:

  • MESMO VALOR:
    • Troca concluída automaticamente
  • DIFERENÇA DE VALOR:
    • Selecione forma de pagamento complementar
    • Quite o valor restante
  • Verifique política de trocas da loja

MÓDULO: TROCA COM CREDEV

1. Processo de Troca Inicial

  1. Realize a troca normalmente no componente PDVFM001
  2. Situação:
    • Cliente devolve produto
    • Escolhe outro de valor inferior
  3. O sistema:
    • Gera crédito CREDEV
    • Vincula ao CPF do cliente

2. Utilização do CREDEV

  1. Em visita posterior:
    • Acesse "MIX DE PAGAMENTOS"
  2. Informe:
    • Valor do CREDEV a utilizar
  3. Selecione:
    • Tipo de pagamento: CREDEV

3. Finalização

  1. Se houver diferença:
    • Selecione pagamento complementar
  2. Conclua a venda normalmente

Observações Importantes:

  • Histórico disponível por CPF
  • Não é transferível

MÓDULO: CASHBACK

1. Geração do Cashback

  1. Após venda:
    • Sistema cria cashback automaticamente
    • Vinculado ao cadastro do cliente

2. Período de Carência

  1. Cliente aguarda:
    • Prazo definido pela campanha
    • Normalmente 7 dias

3. Utilização

  1. Durante nova venda:
    • Sistema exibe saldo disponível
  2. Seleção:
    • Valor a utilizar (total ou parcial)

4. Aplicação

  1. Confirme aplicação
  2. Sistema:
    • Aplica desconto automaticamente
    • Atualiza saldo

5. Finalização

  1. Complete pagamento do restante
  2. Sistema:
    • Gera novo cashback
    • Reinicia ciclo

Observações Importantes:

  • Valor calculado automaticamente
  • Carência varia por campanha
  • Uso opcional pelo cliente
  • Histórico completo disponível

MÓDULO: MOTOR PROMOCIONAL

1. Identificação da Promoção

  1. O sistema verifica automaticamente:
    • Promoções ativas cadastradas
    • Habilita o botão "APLICAR MOTOR PROMOCIONAL"

2. Inclusão de Produtos

  1. Adicione produtos ao carrinho:
    • Via leitor de código de barras
    • OU busca manual
  2. Para produtos participantes:
    • Botão de promoção aparece automaticamente

3. Aplicação da Promoção

  1. Clique em "APLICAR MOTOR PROMOCIONAL"
  2. O sistema:
    • Recalcula valores com desconto
    • Exibe novo total automaticamente

4. Finalização da Venda

  1. Selecione forma de pagamento
  2. Conclua a venda normalmente

Observações Importantes:

  • Promoções válidas apenas para produtos participantes
  • Descontos calculados conforme regras cadastradas
  • Verifique sempre o valor final antes de confirmar
  • Promoções podem ser combinadas conforme configuração

MÓDULO: SUPRIMENTO NO CAIXA

1. Acesso ao Sistema

  1. Acesse o componente PDVFM001 – Venda Varejo
  2. Navegue até o Menu Caixa

2. Seleção da Operação

  1. Selecione a opção "Suprimento"

3. Registro do Valor

  1. Informe:
    • Valor a ser adicionado

4. Informações Complementares

  1. Preencha:
    • Observação (motivo/ref.)

5. Confirmação

  1. Clique em Confirmar
  2. Pressione OK para finalizar

Observações Importantes:

  • Todo suprimento deve ter justificativa
  • Histórico fica registrado com data/hora
  • Verifique valor antes de confirmar
  • Alguns tipos exigem informações extras

MÓDULO: RETIRADA DO CAIXA

1. Acesso ao Sistema

  1. Acesse o componente PDVFM001 – Venda Varejo
  2. Navegue até o Menu Caixa

2. Início da Retirada

  1. Clique no botão "Retirada"

3. Registro das Informações

  1. Preencha:
    • Motivo (obrigatório)
    • Valor a retirar

4. Confirmação Inicial

  1. Clique em "Confirmar"

5. Validação de Segurança

  1. Na janela de validação:
    • Informe sua senha
    • Clique em "Confirmar"

Observações Importantes:

  • Todas as retiradas ficam registradas
  • Motivo deve ser claro e objetivo
  • Valores devem ser comprovados
  • Exige permissão de usuário

MÓDULO: CONTAGEM DO CAIXA

1. Acesso ao Sistema

  1. Acesse o componente PDVFM001 – Venda Varejo
  2. Navegue até o Menu Caixa

2. Início da Contagem

  1. Clique no botão "Contagem"

3. Execução da Contagem

  1. Realize:
    • Contagem física do dinheiro
    • Informe os valores faturados no dia
    • Faça a conferência dos valores informados
  2. Verifique:
    • Possíveis divergências

4. Encerramento

  1. Clique em "Encerrar"
  2. O sistema:
    • Registra valores finais

5. Próximos Passos

  1. Proceda com:
    • Fechamento do caixa

Observações Importantes:

  • Considere todas as movimentações do dia
  • VALOR DE FUNDO deve ser igual ao valor de DINHEIRO
  • O valor de DINHEIRO deve considerar Vendas, Suprimento, Retirada e o valor de caixa do dia anterior

MÓDULO: FECHAMENTO DO CAIXA

1. Preparação

  1. Verifique:
    • Todas vendas registradas
    • Contagem do caixa realizada
    • Nenhuma movimentação pendente

2. Acesso ao Sistema

  1. Acesse o componente PDVFM001 – Venda Varejo
  2. Navegue até o Menu Caixa

3. Início do Fechamento

  1. Selecione "Fechar Caixa"

4. Confirmação

  1. Confirme:
    • Valores exibidos
  2. Clique em "Confirmar"

5. Pós-Fechamento

  1. O sistema:
    • Gera comprovante
    • Bloqueia movimentações
    • Inicia novo ciclo

Observações Importantes:

  • Obrigatório ao final do expediente
  • Irreversível após confirmação

MÓDULO: ENCERRAMENTO DIÁRIO

1. Iniciando o Encerramento

  1. Acesse o componente GERFP010 – Encerramento Diário
  2. Clique em Verificar para:
    • Validar processos pendentes
    • Conferir se há operações em aberto

2. Confirmação do Encerramento

  1. Clique em Confirmar para:
    • Realizar o encerramento diário
    • Avance automaticamente para o próximo dia útil

3. Alteração Manual de Data

  1. Para ajustar a data manualmente:
    • Clique no ícone de calendário
    • Selecione a data desejada

4. Encerramento para Datas Futuras

  1. Se a nova data for 2+ dias à frente:
    • O sistema abrirá automaticamente o componente GERFP081
    • Confirme o dia da semana correspondente
    • Clique em Confirmar para validar

Observações Importantes:

  • O encerramento diário é irreversível após a confirmação
  • Verifique todos os processos pendentes antes de confirmar
  • Para datas futuras, o sistema pode exigir autorização adicional
  • Recomenda-se realizar o encerramento no final do expediente

MÓDULO: CONSULTA DE SALDO DE ESTOQUE – PRDFC014

1. Acesso ao Componente

  1. Acesse o componente PRDFC014:

2. Seleção da Empresa

  1. No campo Empresa:
    • Selecione a unidade desejada

3. Configuração de Saldos e Valores

  1. Configure os parâmetros:
    • Saldo 1: Selecione Físico
    • Valor 1: Selecione Custo médio com imposto
    • Valor 2: Selecione Preço de Venda - Varejo ou Shopping
    • Valor extra: Selecione a primeira opção da lista

4. Agrupamento da Consulta

  1. No campo Agrupamento:
    • Escolha entre Grupo ou Produto

5. Seleção de Produtos

  1. Clique no campo Produto ou Grupo
    • Isso abrirá o componente PRDFF005 – Filtro Geral de Produto
  2. No filtro de produto:
    • No campo Produto por código, digite o código desejado
    • OU utilize o filtro "SOMENTE PRODUTO ACABADO" para listar todos os produtos acabados
  3. Clique em Confirmar para retornar ao PRDFC014

Observações Importantes:

  • Certifique-se de selecionar a empresa correta antes de configurar os filtros
  • Os tipos de saldo e valores podem variar conforme configuração do sistema
  • A opção "SOMENTE PRODUTO ACABADO" filtra TODOS os produtos disponiveis em estoque

MÓDULO: SALDO EM GRADE – PRDFL035

1. Acesso ao Componente

  1. Acesse o componente PRDFL035
  2. Informe a Empresa:
    • Digite manualmente o código
    • OU clique em Seleção... para escolher na lista

2. Configuração Principal

  1. Preencha os parâmetros básicos:
    • Tipo de Saldo: Selecione Físico
    • Tipo de Preço: Escolha entre Preço ou Custo
    • Tipo de Valor: Selecione Venda Varejo ou Shopping

3. Ajustes Adicionais

  1. Marque a opção Listar Saldo
  2. No campo Tipo Valor, selecione:
    • Todos
    • Somente Positivo
    • Zerado
  3. Informe o Tipo de Filtro desejado

4. Seleção de Produtos

  1. Clique no campo Produto:
    • Isso abrirá o componente PRDFF005 – Filtro Geral de Produto
  2. No filtro de produto:
    • No campo Produto por código, digite o código desejado
    • OU utilize o filtro "SOMENTE PRODUTO ACABADO" para listar todos
  3. Clique em Confirmar para retornar ao PRDFL035

5. Execução da Consulta

  1. Para visualizar os resultados:
    • Clique no botão 🔍 (Lupa)
    • OU pressione F4

Observações Importantes:

  • O filtro "SOMENTE PRODUTO ACABADO" lista todos os produtos acabados
  • Os resultados podem ser exportados para Excel se necessário

MÓDULO: BALANÇO – BALFM015

1. Acesso ao Componente

  1. Acesse o componente BALFM015:

2. Criação do Balanço

  1. Gere o número do balanço:
    • No campo Número, pressione Tab para gerar automaticamente
    • Informe uma descrição clara para o balanço

3. Inclusão de Produtos

  1. Clique em "Incluir Produto"
  2. Selecione "NÃO" quando solicitado
  3. Na nova tela exibida:

      • Preencha com 50 (para A)

      • Valor (para análise financeira)
      • OU Quantidade (para análise volumétrica)
    • Agrupador: Escolha entre:
      • Grupo (para análise por categorias)
      • OU Produto (para análise individual)
